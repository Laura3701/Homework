### 1. 输出语句

#### 1.1 document.write

页面打印输出

```javascript
document.write('我是div标签')
document.write('<h1>我是标题</h1>')
```

#### 1.2 alert

弹出警示框

```javascript
alert('hello')
```

#### 1.3 console.log

控制台打印输出

```javascript
console.log('看看对不对')
```

### 2. 输入语句

#### 2.1 prompt

```javascript
prompt('请输入年龄')
```
